    Grid Column


    Core Column memiliki 3 sifat, yaitu float, table cell dan flexbox. Dalam segi penulisan code tidak jauh berbeda. Float menambahkan class (row) Table cell (row-cell) Flexbox (row-flex) untuk membuat column, Float menambahkan (column) Table cell (cell) Flexbox (flex). Untuk membuat kolom sama rata, gunakan class grid, jika menggunakan interior atau inside, maka akan membuat jarak terlalu jauh. Grid class membersihkan kesenjangan itu. Berikut Contoh Penulisannya.

    Beberapa jarak grid yang bisa anda gunakan

    • grid :padding: 0 8px;
    • grid-s :padding: 0 4px;
    • grid-m :padding: 0 16px;
    • grid-l :padding: 0 24px;
    • grid-g :padding: 0 32px;

    grid (Bisa di gunakan untuk semua Column)

    <div class="row grid"> <div class="column grid l6 m6 s6"> Grid </div> <div class="column grid l6 m6 s6"> Grid </div> </div>

    Flexbox (flex)

    Column Flex adalah standard ke 3 untuk pembuatan column Core. Walaupun mempunyai banyak kelebihan, karena Flexbox properti baru dari CSS, yaitu versi ke 3 maka ada 1 browser tidak support pada column ini, yaitu browser Safari. Develover sudah uji coba dan melihat column tetap sama, namun ada kesenjangan jarak yang membuat column menjadi kebawah. Tapi tidak bermasalah, jika anda menggunakan browser populer saat ini, yaitu Mozilla Firefox, Google Chrome dan Opera. Meraka adalah browser yang paling support dan sedikit mempunyai masalah. Bahkan IE yang mempunyai banyak masalah support Column flex.

    Keunggulan dari flexbox adalah tidak menggunakan offset untuk membuat ruang tanpa content. Sifat flexbox berbeda dengan float yang harus menggunakan margin untuk mendorong atau mengkosongkan content. Dengan flexbox dari Core, kita tidak harus menggunakan offset, gunakan code seperti biasanya dan lihat haslinya. Berikut codenya dan lihat hasilnya.
