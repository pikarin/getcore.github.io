    HTML5

    Core sangat menganjurkan untuk memasukan elemen DOCTYPE html. Karena sebagian besar core menggunakan HTML 5.

    <!DOCTYPE hmtl>

    Responsive

    Core adalah framework yang dibuat khusus untuk membuat website responsive. Maka diwajibkan memasukan element ini pada document html.

    <head> <meta name= "viewport" content= "width=device-width, initial-scale=1"> </head>
