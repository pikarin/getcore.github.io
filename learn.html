    Container

    Container

    Digunakan untuk membuat container dengan margin kanan kiri 24px.

    <body> <div class="container"> </div> </body>

    Container Equal

    Digunakan untuk membuat container dengan margin sama sebesar 24px.

    <body> <div class="container-equal"> </div> </body>

    Container Full

    Digunakan untuk membuat container full yaitu margin 0 pada setiap sisi.

    <body> <div class="container-full"></div> </body>

    Inside

    Digunakan untuk membuat margin kanan dan kiri sebesar 16px. Ini digunakan untuk content agar tidak terlalu dekat dengan dinding margin container.

    inside : margin: 0px 16px;
    inside-s : margin: 0px 8px;
    inside-m : margin: 0px 24px;
    inside-l : margin: 0px 32px;
    inside-g : margin: 0px 64px;
    (default) : normal
    (s) : Small
    (m) : Medium
    (l) : Large
    (g) : Gigant
    Kenapa marginnya 0px 8px? Kok ada dua element?

    Core akan membari tahu teknik penulisan diatas

    Jika satu element, artinya margin atas, kanan, bawah, kiri itu sama, berikut contohnya
    margin : 8px

    Jika dua element, maka element yang pertama adalah atas bawah dan element yang kedua kanan kiri.
    margin : 0px 8px | Berarti margin atas bawah 0px, kanan kiri 8px

    Jika 3 element, maka element yang pertama adalah atas, yang kedua kanan kiri, dan yang ketiga adalah bawah
    margin : 0px 8px 16px | Berarti margin atas 0px, margin kanan kiri 8px dan margin bawah 16px.

    Dan yang terakhir jika 4 element, berarti setiap sisi di atur marginya.
    margin : 0px 8px 16px 32px | Berarti margin atas 0px kanan 8px bawah 16px kiri 32px.

    Ingat polanya : atas kanan bawah kiri
    <body> <div class="containter"> <div class="inside"></div> </div> </body>

    Interior

    Sama seperti inside, interior digunakan untuk membuat content tidak terlalu dekat dengan margin container, bedanya, interior adalah padding sebeser 16px. Penulis lebih rekomendasi menggunakan interior untuk membuat content terlihat baik.

    interior : padding: 0px 16px
    interior-s : padding: 0px 8px
    interior-m : padding: 0px 24px
    interior-l : padding: 0px 32px
    interior-g : padding: 0px 64px
    <div class="container"> <div class="interior"></div> </div>

    Navbar
    Navbar versi sebelumnya di hilangkan karena alasan tidak memenuhi standarisasi Flamecore Organization. Maka navbar Navoon hadir dengan dengan membawa perbedaan dari navbar biasanya. Responsive di semua ukuran. Menggabungkan classic dan modren pada designnya. Sangat simple dan multi element. Pada versi ini kita bisa membuat sidebar dan navbar sekaligus, juga kita dapat membuat keperluan lain lain di icon   .

    Di awali dari sidebar. Animation sidebar sudah mempunyai default 10 link. Maka animasi berikutnya tidak memiliki animation untuk mengatasinya, masukan code berikut di style.

    .navoon-aside li{ position:relative!important;-webkit-animation:navoonAside 0.6s!important;animation:navoonAside 0.6s!important} @-webkit-keyframes navoonAside{from{left:-300px} to{left:0}} @keyframes navoonAside{from{left:-300px} to{left:0;} }

    !important berguna untuk mendahulukan animation. Karena navbar navoon sudah memiliki animation sendiri. Jadi gunakan element tersebut untuk menukar peran element.

    Fungsi click pada navoon aside di buat dengan javascript, maka kita harus meletakan scriptnya dibawah body

    <body> </body> <script type="text/javascript" src="js/navoon.js"></script>

    Kedua adalah logo. Pastikan logo di letakan pada element kedua, karena tag li sudah di atur pada child kedua untuk logo. Karena navbar navoon memiliki hover, untuk navbar logo juga akan terkana effect hover, jangan khawatir, core sudah membuatnya transparent. Ukuran logo 185px x 40px dan tentunya harus berekstensi png. CSS sudah di atur : Max-width: 100% dan height: 40px. Padding pada child logo sudah di reset 0, maka jika logo tidak sama rata, gunakan padding top untuk meratakannya. Tapi tidak perlu khawatir, padding-top sudah di default 9px untuk meratakannya.

    .navoon li:nth-child(2) a{ padding:0; } .navoon li:nth-child(2):hover a{ background-color: transparent; } .navoon-logo{ max-width: 100%; height:40px; padding-top:9px; }

    Ketiga adalah icon    untuk menambahkan element yang kita butuhkan. icon ini sudah memiliki default untuk navbar, map, link, note, dan about. Animation Effect hanya terdapat 5 element, dan penulis sangat tidak menganjurkan lebih dari 5 element karena tidak akan responsive. Karena width dari masing masing child sudah di atur 20%.

    <div class="navoon-tab"> <button class="tablinks" onclick="openContent(event, 'content1')" id="defaultOpen"> <i class="fa fa-ellipsis-h"></i></button> <button class="tablinks" onclick="openContent(event, 'content2')"> <i class="fa fa-language"></i> </button> <button class="tablinks" onclick="openContent(event, 'content3')"> <i class="fa fa-map-marker"></i> </button> <button class="tablinks" onclick="openContent(event, 'content4')"> <i class="fa fa-credit-card-alt"></i> </button> <button class="tablinks" onclick="openContent(event, 'content5')"> <i class="fa fa-book"></i> </button> </div>

    id="defaultOpen" di gunakan untuk menampilkan element pertama kali di buka, jika tidak ingin menampilkan element ketika pertama kali di buka, hapus saja element tersebut.

    (event, 'content1') 'content1' adalah id dan harus sama dengan id tabcontent agar script javascript berjalan.

    .tabcontent { display: none; padding: 0; margin: 0; z-index: 8!important; border-left:1px solid #ddd; overflow: auto; height:300px; } @media (max-width:600px){ .tabcontent { border-left: none!important; border-bottom:1px solid #ddd; } }

    Jika ukuran height terlalu kecil, tambah ukuran height. Dan tidak di anjurkan menghapus height karena content body pada ukuran smartphone akan tertutup jika element yang kita masukan di class tabcontent terlalu panjang.

    pastikan pada tabcontent id sama dengan tablinks

    <div id="content1" class="tabcontent"> <nav class="nav" id="cNav"> <a href="index.html">Flamecore</a> <a href="learn.html" target="_blank">Learn</a> <a href="column.html" target="_blank">Column</a> <a href="color.html" target="_blank">Color</a> <a href="author.html" target="_blank">Author</a> <a href="javascript:void(0);" class="toggle" onclick="cToggle()"><i class="fa fa-list"></i></a> </nav> </div>

    Jika ingin membuat element baru selain defaut navbar navoon, maka ganti id tabcontent dan id tablink. Untuk mengatur height otomatis bisa menambahkan element di style pada class tabcontent menjadi auto, scroll atau lain lain.

    <div id="content1" class="tabcontent" style="height:auto"></div>

    Layout

    class layout, mempunyai default max width 1500px digunakan pada tag body atau tag yang ingin dicostumisasi. Memungkinkan layout akan berada di tengah walaupun ukuran browser diperkecil. pada umumnya mengecilkan browser adalah (ctrl) + (-).

    layout max-width: 1500px margin: auto
    layout-s max-width: 800px margin: auto
    layout-m max-width: 1200px margin: auto
    layout-l max-width: 1600px margin: auto
    layout-g max-width: 2000px margin: auto

    Penerapan class layout pada body

    <body class="layout"> </body>

    Penerapan class layout pada tag yang ingin di modifikasi

    <body> <div class="layout"> </div> </body>

    Modifikasi Layout

    <body> <div class="layout" style="max-width: 700px"> </div> </body>
