Learn Core

Intro

Core adalah karya anak negeri bernama Andika Chamberlin. Untuk itu gunakanlah. Berarti kita bangga dengan karya indonesia. Seharusnya kita juga sadar, dimana kita berasal dan akan mendukung setiap karya yang baik untuk indonesia. Merdeka!!!

Sejarah

Alangkah baiknya, sebelum belajar core, kita harus tahu kenapa framework core bisa di ciptakan oleh orang aneh bernama andika chamberlin. Alasan Pertama : Dika nama panggilannya, adalah seorang yang sangat individual, kebiasannya itu adalah suka berhayal tentang mimpi mimpi, dan salah satunya yang sudah terwujud adalah framework core. Sejujurnya ia ingin membuat framework php, framework javascript framework SQL dan yang paling dia inginkan menciptakan exstensi program baru, yang ada di benaknya saat ini adalah extensi file (.impact) yaitu bahasa pemograman khusus buat orang indonesia :), agar kita tidak susah susah mengerti syntax bahasa inggris :). Klimaksnya ia ingin membuat Operating system yang akan membawa indonesia lebih di segani oleh negara lain mungkin dengan menciptakan OS baru yang terkenal di indonesia khususnya, kita sebagai warga indonesia tidak dipandang lemah dalam segi teknologi. Saat ini nama OS yang ada di impiannya adalah Magnum atau Diamon.

Alasan kedua: si Dika ini orangnya sangat tidak suka menggunakan barang yang hanya tinggal pakai, di pikirannya adalah, ("percuma memakai tapi tidak tahu apa apa"), dari situ ia belajar mati matian hingga jatuh sakit (yang sering sakit hati) untuk terus belajar bahasa CSS lalu Sass SCSS dan lain lain. Sekarang, ia telah faham dengan CSS, dan dia telah membuat Core unggul pada sistem grid, yaitu sistem grid menggunakan float, table cell, flexbox dalam kondisi grid yang berbeda beda : 2, 4, 8, 12, 16, 24.

Alasan ketiga: Dika ingin sekali bermanfaat bagi orang lain, dika ingin membantu orang lain, dan dika ingin mempunyai banyak teman baik. Doakan dika masuk surga. Amiin. Semoga framework core bermanfaat buat orang banyak amiin.

Saatnya Belajar GO!
