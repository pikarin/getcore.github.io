Display

Inline

Make the display into inline which means sideways, width depends on Content. Commonly inline tags are the < span >

Core Balloon <span class="inline" style="border:1px solid #f1f1f1">Core</span> <span class="inline" style="border:1px solid #f1f1f1">Balloon</span>

Block

Makes a full 100% display, meaning the next content is below. Tag < div > Is one of the tags that are block.

Core
Balloon
<div class="block" style="border:1px solid #f1f1f1;">Core</div> <div class="block" style="border:1px solid #f1f1f1">Balloon</div>
