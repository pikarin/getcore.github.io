Dropdown

Dropdown Core tidak memiliki fungsi untuk close dropdown content dengan mengclick dimana saja. Jika menginginkan click dimana saja di anjurkan untuk membuatnya manual. Selebihnya dropdown core memanjakan anda dengan dropdown active dan smooth tanpa jQuery. Dropdown Core menawarkan 3 jenis sifat, yaitu basic, fade, dan swing.

Drodown Basic

Dropdown basic default dengan arah kanan dan memiliki caret transform berputar untuk defaultnya. Ketiga jenis Dropdown memiliki 3 Transform caret ketika di click. caret-down dan caret up circle, card-v (vertikal) dan card h(horizontal)

Function Javascript basic dropdown

function basicDropdown() { document.getElementById("basicDropdownContent").classList.toggle("basic-show-dropdown"); document.getElementById("basicActiveDropdown").classList.toggle("basic-active-dropdown"); document.getElementById("changeBasicToggle").classList.toggle("caret-up-circle"); document.getElementById("changeBasicToggle").classList.toggle("caret-down-circle"); }

Untuk mengatur active basic dropdown, jika menginginkan warna text cukup membuatnya color : (warna) jika ingin active background maka gunakan background-color : (warna).

.basic-active-dropdown{ background-color: #ddd; }

Ada 3 pilihan effect transform caret dropdown

caret-down-circle & caret-up-circle
caret-down-card-v & caret-up-card-v
caret-down-card-h & caret-up-card-h

Untuk mengganti effect caret pada class

class="caret-down-circle" Menjadi class="caret-down-card-v"

dan ganti function javascript

document.getElementById("changeBasicToggle").classList.toggle("caret-up-card-v"); document.getElementById("changeBasicToggle").classList.toggle("caret-down-card-v");

Dropdown Swing

Dropdown Swing dibangun dengan visibility, dengan properti itu, effect transition seperti ayunan dapat di lakukan. Karena effect swing di bangun dengan visibility, maka untuk mengatur supaya click terbuka semua, kita di anjurkan menganti ukuran height, kira kira berapa height untuk memunculkan keseluruhan Dropdown Content. Bisa juga menggunakan height 100%, yaitu pada 1 tag a di beri nilai 100%, untuk membuka dropdown content dengan effect swing, berarti jika terdapat 7 tag a, maka kita membutuhkan height 700%.

.show-dropdown-swing { visibility: visible; height: 380px; overflow: hidden; }
