Layout

Class layout, has default max width 1500px used in body tags or tags that want to be customized. Allows the layout to be centered even if the browser size is minimized. Generally shrink the browser is (ctrl) + (-).

layout max-width: 1500px margin: auto
layout-s max-width: 800px margin: auto
layout-m max-width: 1200px margin: auto
layout-l max-width: 1600px margin: auto
layout-g max-width: 2000px margin: auto

Implementation of body layout class

<body class="layout"> </body>

Implementation of the layout class in the tag you want to modify

<body> <div class="layout"> </div> </body>

Modify Layout

<body> <div class="layout" style="max-width: 700px"> </div> </body>
