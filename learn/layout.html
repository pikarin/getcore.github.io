Layout

class layout, mempunyai default max width 1500px digunakan pada tag body atau tag yang ingin dicostumisasi. Memungkinkan layout akan berada di tengah walaupun ukuran browser diperkecil. pada umumnya mengecilkan browser adalah (ctrl) + (-).

layout max-width: 1500px margin: auto
layout-s max-width: 800px margin: auto
layout-m max-width: 1200px margin: auto
layout-l max-width: 1600px margin: auto
layout-g max-width: 2000px margin: auto

Penerapan class layout pada body

<body class="layout"> </body>

Penerapan class layout pada tag yang ingin di modifikasi

<body> <div class="layout"> </div> </body>

Modifikasi Layout

<body> <div class="layout" style="max-width: 700px"> </div> </body>
