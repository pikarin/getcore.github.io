Container

Container

To create a container with left margin 24px left.

<body> <div class="container"> </div> </body>

Container Equal

Used to create a container with the same margin of 24px.

<body> <div class="container-equal"> </div> </body>

Container Full

Used to create a full container that is a margin of 0 on each side.

<body> <div class="container-full"></div> </body>

Inside

Used to create a right and left margin of 16px. This is used for content not too close to the wall of margin container.

inside : margin: 0px 16px;
inside-s : margin: 0px 8px;
inside-m : margin: 0px 24px;
inside-l : margin: 0px 32px;
inside-g : margin: 0px 64px;
(default) : normal
(s) : Small
(m) : Medium
(l) : Large
(g) : Gigant
<body> <div class="containter"> <div class="inside"></div> </div> </body>

Interior

Just like inside, the interior is used to make the content not too close to the margin container, the difference, the interior is padding 16px shift. More authors recommend using the interior to make the content look good.

interior : padding: 0px 16px
interior-s : padding: 0px 8px
interior-m : padding: 0px 24px
interior-l : padding: 0px 32px
interior-g : padding: 0px 64px
<div class="container"> <div class="interior"></div> </div>

Deep

Make a margin equal to 16px

deep : margin: 16px
deep-s : margin: 8px
deep-m : margin: 24px
deep-l : margin: 32px
deep-g : margin: 64px
<div class="container"> <div class="deep"></div> </div>

Equal

Make padding equal to 16px

equal : padding: 16px
equal-s : padding: 8px
equal-m : padding: 24px
equal-l : padding: 32px
equal-g : padding: 64px
<div class="container"> <div class="equal"></div> </div>
