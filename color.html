Untuk membuat warna text berwarna, masukan class (warna). Berikut contoh warna code


<span class="crimson">Crimson</span>

Crimson

Untuk membuat text hover warna, gunakan class text-hover-(warna). Berikut codenya


<span class="text-hover-crimson">Crimson</span>
Hover Crimson

Untuk membuat warna background, gunakan class bg-(warna).Berikut codenya. 


<div class="bg-crimson">bg-crimson</div>

bg-crimson

Untuk membuat hover background, gunakan class hover-(warna). Berikut codenya. 


<div class="hover-crimson">hover-crimson</div>

Tahukah Kamu ?

Warna dasar itu bukan merah, kuning, hijau, biru, ataupun hitam dan putih. Tetapi RGB (Red, Green, Blue). Merah, hijau dan biru.
Apakah kamu tahu, kenapa warna lampu kendaraan pada umumnya berwarna kuning? Karena warna kuning tidak kehilangan warnanya jika di gabungkan dengan warna putih. Jelasnya warna putih seperti kehilangan warnanya.
Apakah kamu tahu, kenapa lambang cinta (love) selalu berwarna merah? Karena warna merah melambangkan keberanian, yang berarti kita berani mencintai tanpa dicintai, berani merelakan, berani berkorban, berani mengalah, berani menyatu, berani untuk membangun kebersamaan sampai akhir hayat. Dan keberanian lain-lain dalam versi keberanianmu dalam cinta.
